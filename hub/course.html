> LEARNING PATH // 5 MODULES

Ethical Hacking Course

A structured path from networking basics to bug bounty hunting. Each module builds on the last.

MODULE 01 Networking Fundamentals ~3 weeks
Everything runs on networks. You cannot attack or defend what you don't understand at the packet level.
  • OSI model layers and TCP/IP stack
  • IP addressing, subnetting, CIDR notation
  • DNS, DHCP, ARP — how name resolution works
  • TCP handshake, UDP, ICMP
  • Packet capture with Wireshark — reading pcap files
  • Common protocols: HTTP, FTP, SSH, SMTP
MODULE 02 Reconnaissance & Scanning ~3 weeks
Map the target before touching it. Passive OSINT leaves no trace; active scanning leaves logs.
  • OSINT techniques: WHOIS, Shodan, Google dorks
  • Subdomain enumeration with amass / subfinder
  • Nmap port scanning — flags, timing, scripts (NSE)
  • Service and version fingerprinting
  • Vulnerability scanning with Nikto and OpenVAS
  • Building a target profile from public data
MODULE 03 Exploitation Techniques ~4 weeks
Understand how real attacks work by replicating them in lab environments — never on production systems without written scope.
  • Metasploit framework — modules, payloads, sessions
  • Common CVEs and how to look them up (NVD, ExploitDB)
  • Buffer overflows — theory and practice on lab binaries
  • Web attacks: SQLi, XSS, CSRF, file inclusion
  • Password attacks: dictionary, brute-force, hash cracking
  • Setting up a home lab (VMs, Vulnhub, HackTheBox)
MODULE 04 Post-Exploitation & Reporting ~3 weeks
Getting in is half the job. Understanding impact and communicating it clearly is what clients actually pay for.
  • Privilege escalation on Linux and Windows
  • Lateral movement — pass-the-hash, pivoting
  • Persistence mechanisms (for lab / CTF contexts)
  • Covering tracks — log awareness and forensics basics
  • Writing a pentest report: executive summary to PoC
  • CVSS scoring and risk rating findings
MODULE 05 Bug Bounty Hunting ~4 weeks + ongoing
Apply everything on real-world programs. Bug bounty turns ethical hacking into a paid skill you can build a career on.
  • Platforms: HackerOne, Bugcrowd, Intigriti — reading scopes
  • OWASP Top 10 — the core checklist for web targets
  • Burp Suite Pro workflow for web app testing
  • SSRF, IDOR, broken access control — high-payout classes
  • Writing a quality disclosure report
  • Building a reputation and moving to private programs

// Recommended certifications

After completing the learning path, these certs validate your skills to employers:

  • CompTIA Security+ — entry-level, vendor-neutral, widely recognized
  • CEH (Certified Ethical Hacker) — EC-Council, good for corporate roles
  • OSCP (Offensive Security Certified Professional) — hands-on, industry gold standard
  • eJPT (eLearnSecurity Junior Penetration Tester) — great first cert, affordable
